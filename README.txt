Данная программа моделирует простую однопоточную систему массового обслуживания.
Интерфейс даёт пользователю возможность задать параметры работы системы (такие как скорость обработки заявок, частота появления
заявок, закон распределения появления заявок и тп).
Также интерфейс предоставляет статистику по работе СМО и визуализирует текущую очередь.
